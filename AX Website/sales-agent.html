Sales Agent

디지털보험 탐색부터 가입 완료까지 고객과 동행하며, 설명 부족과 이해 부족 때문에 생기는 이탈을 줄이는 AX 과제입니다.

3개의 Agent가 각각 탐색, 이해, 청약완주 단계를 맡고, 이 흐름 전체가 하나의 Sales Agent 경험으로 연결됩니다.

Agent 01 맞춤설계 Agent

고객 조건을 바탕으로 추천안 3개를 빠르게 제안합니다.

Agent 02 설계이해 Agent

어려운 보장 내용을 고객 언어로 풀어 설명합니다.

Agent 03 청약완주 Agent

막히는 단계에서 이어하기와 도움말로 완주를 돕습니다.

01

Journey

3개의 Agent가 각 병목을 맡고,
합쳐서 하나의 Sales Agent 경험을 만듭니다.

고객은 한 번에 긴 보험 가입 절차를 통과하는 것이 아니라, 단계마다 다른 불안을 겪습니다. Sales Agent는 이 불안을 3개의 전문 Agent로 나눠 해결하고, 고객 입장에서는 끊기지 않는 하나의 도움 경험으로 느껴지도록 설계합니다.

01

맞춤설계 Agent

프로토타입 보기

탐색 단계에서 "나에게 맞는 상품이 무엇인지"를 바로 좁혀주는 Agent

고객의 나이, 성별, 지불 가능한 월 보험료, 집중 보장 항목을 받아 고객별 추천 설계안 3개를 제안합니다. 복잡한 셀프 계산이나 수많은 상품 비교를 먼저 요구하지 않고, 고객이 바로 검토 가능한 후보군을 보여주는 것이 핵심입니다.

즉, 탐색 초반의 피로를 줄이고 보험료 확인 단계까지 더 빠르게 진입하게 만들어 가입 여정의 첫 병목을 줄입니다.

  • #추천안 3개 제안
  • #셀프계산 단계 축소
  • #보험료 확인 유도
02

설계이해 Agent

프로토타입 보기

이해 단계에서 "무슨 뜻인지 모르겠다"는 불안을 풀어주는 Agent

보험 용어, 보장 항목, 제한사항, 추천 근거를 고객이 이해하기 쉬운 언어로 다시 설명합니다. 고객이 설계를 스스로 조정하고 싶어도 용어를 이해하지 못해 멈추는 상황을 줄이고, 납득 가능한 판단을 돕는 역할을 맡습니다.

결국 이 Agent는 단순 안내가 아니라, 가입 여부 최종 판단 직전에 생기는 이해 장벽을 낮춰 긍정적인 결정 가능성을 높이는 장치입니다.

  • #보험용어 쉬운 설명
  • #보장 의미 해석
  • #추천 근거 안내
03

청약완주 Agent

TBD

가입 단계에서 "여기서 막히면 포기한다"는 순간을 넘기게 돕는 Agent

청약 과정 중 입력, 인증, 동의, 제출 단계에서 막히는 부분의 도움말을 제공하고, 이탈 시 임시저장이나 이어서 진행하는 방법을 안내합니다. 고객이 중간에 흐름을 잃지 않도록 인내심 있게 완주를 독려하는 것이 중요합니다.

이 Agent는 단순 FAQ가 아니라, 실제 청약 완결까지 고객을 끌고 가는 마지막 완주 보조 장치로 작동합니다.

  • #막힘 단계 도움말
  • #임시저장/이어하기
  • #완주율 개선
