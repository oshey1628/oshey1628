# AI Agent 보험 추천 프로토타입 테스트케이스

## 1. 목적
- 1단계 상품상세페이지 → 바텀시트 → 로딩 → 추천설계 3안 → 설계 상세 팝업 → 2단계 계산결과화면 이동 흐름이 정상 동작하는지 확인한다.
- 입력값(나이대/성별/월보험료/주요 보장)에 따라 추천 설계안과 상세 팝업 내용이 연관성 있게 노출되는지 확인한다.
- `직접설계할게요`, `상세설계 확인/변경` 선택 시 2단계 계산결과화면으로 정상 이동하는지 확인한다.
- 이동 후 2단계 계산결과화면의 보험료가 앞 단계에서 선택한 추천 설계 보험료와 동일한지 확인한다.

## 2. 테스트 범위

### 2-1. 1단계 입력 조건

#### 나이대
- A1: 20대 이하
- A2: 30대
- A3: 40대
- A4: 50대 이상

#### 성별
- G1: 남
- G2: 여

#### 월 보험료
- P1: 3~5만원
- P2: 5~10만원
- P3: 10~15만원
- P4: 15만원 이상

#### 가장 중요한 보장
- N1: 암
- N2: 뇌심장
- N3: 치매
- N4: 골고루

### 2-2. 추천 설계안
- R1: 추천 1
- R2: 추천 2
- R3: 추천 3

## 3. 조합 수
- 기본 입력 조합: `4(나이대) x 2(성별) x 4(월보험료) x 4(보장)` = `128개`
- 추천 설계안 선택 검증 포함: `128 x 3` = `384개`
- `직접설계할게요` 이동 검증 포함 시 추가 `128개`

## 4. 공통 기대결과

### 4-1. 1단계 상품상세페이지
- 생년월일 입력 시 해당 나이대가 내부적으로 정상 반영된다.
- 성별 토글 선택이 정상 반영된다.
- `보험료 확인하기` 선택 시 바텀시트가 정상 노출된다.

### 4-2. 2단계 바텀시트
- 월 보험료 4개 선택지 중 1개를 선택할 수 있다.
- 다음 단계로 자연스럽게 이동된다.

### 4-3. 3단계 바텀시트
- `암`, `뇌,심장`, `치매`, `골고루` 중 1개를 선택할 수 있다.
- 선택 후 로딩 애니메이션과 문구가 정상 노출된다.

### 4-4. 추천 설계안 화면
- 추천 설계안 3개가 모두 노출된다.
- 추천 설계안 문구/타이틀/설명/보험료가 깨지지 않는다.
- `직접설계할게요` 선택 시 2단계 계산결과화면으로 이동한다.

### 4-5. 추천 설계안 선택 후 팝업
- 선택한 추천안에 맞는 타이틀이 노출된다.
- 선택한 보장 성격과 연관된 보장항목이 노출된다.
  - 암 선택 시 암 관련 보장이 우선 노출되는가?
  - 뇌심장 선택 시 뇌/심장 관련 보장이 우선 노출되는가?
  - 치매 선택 시 치매/간병 관련 보장이 우선 노출되는가?
  - 골고루 선택 시 특정 보장 쏠림 없이 균형형으로 보이는가?
- `상세설계 확인/변경` 선택 시 확인 팝업을 거쳐 2단계 계산결과화면으로 이동한다.
- `가입시작` 선택 시 3단계 청약시작 화면으로 이동한다.

### 4-6. 2단계 계산결과화면
- 진입 시 화면이 정상 노출된다.
- 상단 보험료와 CTA 영역이 깨지지 않는다.
- 앞 단계에서 선택한 추천안의 보험료와 동일한 금액이 반영된다.

## 5. 테스트 ID 규칙
- 기본 시나리오 ID: `A?-G?-P?-N?`
- 추천안 선택 검증 ID: `A?-G?-P?-N?-R?`
- 직접설계 이동 검증 ID: `A?-G?-P?-N?-D`

예시:
- `A3-G1-P2-N1`: 40대 / 남 / 5~10만원 / 암
- `A3-G1-P2-N1-R1`: 위 조건에서 추천 1 선택 후 팝업 검증
- `A3-G1-P2-N1-D`: 위 조건에서 `직접설계할게요` 선택 후 2단계 이동 검증

## 6. 전체 실행 규칙

아래 128개 기본 입력 조합 각각에 대해 동일하게 수행한다.

1. 1단계에서 생년월일 입력
2. 성별 선택
3. `보험료 확인하기` 선택
4. 월 보험료 선택
5. 주요 보장 선택
6. 추천 설계안 3개 노출 확인
7. `R1`, `R2`, `R3` 각각 선택 후 팝업 검증
8. 각 팝업에서 `상세설계 확인/변경` 선택 시 2단계 계산결과화면 이동 여부 확인
9. 2단계 계산결과화면 보험료가 선택한 추천안 보험료와 동일한지 확인
10. 추천 설계안 화면에서 `직접설계할게요` 선택 시 2단계 계산결과화면 이동 여부 확인

## 8. 추천 설계안 검증 체크리스트

각 기본 입력 조합마다 아래 4개를 모두 확인한다.

### 8-1. R1 선택
- 추천 1 팝업이 열린다.
- 타이틀이 선택한 보장 맥락과 맞는다.
- 보장항목이 선택한 보장과 연관성 있게 우선 노출된다.
- `상세설계 확인/변경` 선택 시 2단계 계산결과화면으로 이동한다.
- 2단계 계산결과화면 보험료가 추천 1 보험료와 동일하다.

### 8-2. R2 선택

- 타이틀/보장항목이 균형형 맥락과 맞는다.

- 2단계 계산결과화면 보험료가 추천 2 보험료와 동일하다.

### 8-3. R3 선택
- 추천 3 팝업이 열린다.
- 타이틀/보장항목이 실속형 맥락과 맞는다.
- `상세설계 확인/변경` 선택 시 2단계 계산결과화면으로 이동한다.
- 2단계 계산결과화면 보험료가 추천 3 보험료와 동일하다.

### 8-4. 직접설계
