너는 시니어 프로덕트 디자이너이자 화면 기획 전문가야. 
'삼성생명 민원 지식봇'화면을 기획해 docs/screen_spec.md로 정리해줘. 

포함: 
- 정보구조(IA): 무엇을 위에, 무엇을 접을지
- 핵심 흐름: 질문 입력 -> 로딩 -> 답변+근거 -> 담당자 홧인
- 화면 상태 5종: 초기 / 입력중 / 로딩 / 결과 / 오류,범위 밖
- 컴포넌트: 질문 입력, 실행 버튼, 답변 카드, 근거 FAQ 리스트, '담당자 확인 필요' 뱃지, 면책 배너
- UX 라이팅: 버튼, 플레이스홀더, 빈 상태, 오류문구(한국어, 신뢰감)
- 접근성: 색 대비,키보드,읽기 순서 / 모바일 우선

기획서를 먼저 보여주고, 합의되면 그대로 index.html 화면뼈대를 만들어줘(검색·답변로직은비움). 개인정보 비슷한 숫자 입력은 막아줘. 로컬 미리보기로.
