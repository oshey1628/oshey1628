(목적)
아래 프로세스가 단계적으로 수행되는 Html 페이지의 프로토타입을 만들어줘
전체적인 프로세스는 아래를 참고해서 이해하고,
목적은 팩건강보험 상품상세페이지를 확인한 고객이, 본인의 나이/성별/낼수있는보험료/니즈에 맞게 3가지 조합을 추천해서 설계해주는 AI Agent 를 제공하고 싶어 
전체적인 프로세스는 첨부의 피피티파일 와이어프레임을 참고해

(프로세스)
(1단계. 상품상세페이지)
1.팩건강보험 상품 상세페이지에서 고객이 생년월일과 성별을 입력하고 '보험료 확인하기' 버튼을 터치
2. 바텀시트방식으로 하단에서 바텀시트가 올라올라옴
3. 올라온 바텀시트에서 고객이 낼수 있는 월보험료 수준을 질문하고(예시: 매월 보험료는얼마정도 생각하세요?), 박스형태의 선택지로 제시한 후 고객이 하나를 터치 (선택박스는 3~5만원, 5~10만원, 10만원 이상)
4. 선택된 값을 기억한 후, 새로운 질문으로 '어떤 보장을 더 우선시 하는지'를 질문하고 (예시: 어떤 상황을 가장 대비하고 싶으세요?) 4개의 선택지를 제시함
- 암에 대한 걱정이 제일 커요
- 뇌,심장 혈관질환이 가장 걱정돼요
- 치매가 정말 염려돼요
- 골고루 대비하고 싶어요
5. 로딩에니메이션 노출 (로딩에니메이션에서 앞 단계에서 입력한 나이,성별,보험료,중요보장 에 맞게 멘트 제공, '월 5~10만원내에서 40대 남성에게 암보장에 딱맞게 설계중이에요'
6. 로딩에니메이션 후 3,4단계에서 선택한 값에 따라 3개의 추천설계(41개의 보장항목을 선택한 보험료와 중요 보장항목에 따라 조합한 세트)를 선택카드 형태로 제시(한줄 추천근거, 보험료 함께 노출하고 보험료는 볼드체로 눈에 잘잘보이게)
7. 선택한 ,4. 3개의 추천설계중 하나를 선택
8. 팝업을 통해 선택한 조합으로 AI설계가 적용되었고, 자세한 설계내용을 확인한후 바로 가입해보세요 안내하기 (버튼은 '보장확인', '바로가입') 
9. 보장확인 시 해당 선택조합의 보장항목을 반영하여 (2단계. 계산결과 화면)으로 이동
10. '바로가입' 선택시 (청약프로세스 1단계화면) 으로 이동

(2단계. 계산결과화면)
1.계산결과화면을 스크롤하면서 하단의 세부 보장항목을 열어 실제 세팅된 값을 확인가능
(첨부하는 '계산결과화면' 이미지를 그대로 노출하고 스크롤 가능하도록 함)
2.'가입하기' 선택시 (청약프로세스 1단계화면) 으로 이동
3.헤더에서 < 선택 시 (1단계. 상품상세페이지)의 6번단계로 이동 (추천설계 카드가 보여지는 바텀시트)

(3단계. 청약프로세스 1단계화면)
1.첨부하는 '청약프로세스1단계' 이미지를 그대로 노출하고 스크롤 가능하도록 함
2.헤더에서 < 선택 시 (2단계. 계산결과화면) 으로 이동

일단, 1단계부터 하나씩 우선 만들고, 나와 확인하면서 문제가 없을 때, 다음단계 화면개발될 수 있도록 순차적으로 진행하자

제작할때 폰트는 noto sans 로 적용해주고,
키컬러는 파란색(밝은) + 화이트 조합으로 해줘
