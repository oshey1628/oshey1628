# AI assistant for Pack 인계 메모

## 현재 목표
- 팩건강보험 상품상세페이지에서 고객의 생년월일/성별/월 보험료/우선 보장 니즈를 받아 AI Agent가 3가지 추천 설계를 제안하는 HTML 프로토타입을 단계적으로 제작한다.
- 현재는 1단계 상품상세페이지 흐름만 제작했다.

## 현재 산출물
- `index.html`: 1단계 HTML 프로토타입
- `assets/product-detail.jpg`: 1단계 상품상세페이지 참고 이미지
- `assets/process-reference.png`: 전체 프로세스 참고 이미지

## 구현된 1단계 흐름
1. 상품상세페이지 이미지 노출
2. `보험료 확인하기` 영역 터치
3. 바텀시트 노출
4. 월 보험료 선택
   - 3~5만원
   - 5~10만원
   - 10만원 이상
5. 우선 보장 니즈 선택
   - 암
   - 뇌·심장 혈관질환
   - 치매
   - 균형
6. 선택값 기반 로딩 문구 노출
7. 추천설계 3개 카드 노출
8. 추천 카드 선택 시 `보장확인` / `바로가입` 팝업 노출

## 최근 수정사항
- 전체 폰트 렌더링에 안티앨리어싱 옵션 적용
- 바텀시트 선택박스 텍스트를 18px로 키움
- 바텀시트 선택박스 텍스트 bold 제거
- 선택 후 다음 바텀시트 콘텐츠가 오른쪽에서 슬라이딩되어 나타나는 애니메이션 적용

## 아직 연결하지 않은 부분
- `보장확인`: 2단계 계산결과화면으로 연결 예정
- `바로가입`: 3단계 청약프로세스 1단계화면으로 연결 예정
- 현재 두 버튼은 임시 안내 토스트만 노출한다.

## 다음 작업 추천
